# VGSales
A Tableau and SQL project that involves answering questions about video game sales from a data set. 
data source: https://www.kaggle.com/datasets/willianoliveiragibin/video-game-sales-analyze


Questions:
1. What are the top 5 publishers by sales?
2. What are Nintendo's top genres?
3. Which region yields the most sales for Nintendo?
4. What were Nintendo's top years of sales?
